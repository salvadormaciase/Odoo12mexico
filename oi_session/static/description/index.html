Inactive Sessions Timeout

"Optional Timeouts for Inactive Sessions"

This module allows administrator to set timeout for inactive sessions.

Description

Session timeout is a security and resource management feature that automatically logs a user out of system. The inactivity timeout is a configurable period during which a user can be inactive (that is, not interact with the system in any way) without any impact on their session. This module helps you to control when an inactive user session expires, also allows you to configure the default session timeout hours of inactivity. Odoo will automatically delete the inactive sessions after timeout.
